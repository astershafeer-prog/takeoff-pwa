# TakeOff Member Directory PWA - Demo

## Quick Start

1. Install Flask:
```bash
pip install flask --break-system-packages
```

2. Run the app:
```bash
python app.py
```

3. Open in browser:
```
http://localhost:5000
```

## Test on Phone

### Option 1: Local Network (Same WiFi)
1. Find your computer's IP address:
   - Windows: `ipconfig` (look for IPv4)
   - Mac/Linux: `ifconfig` or `ip addr`
2. On your phone, open: `http://YOUR_IP:5000`

### Option 2: Deploy Online (Render.com - Free)
1. Create account at render.com
2. New Web Service
3. Connect this folder
4. Deploy
5. Get public URL (works on any phone)
